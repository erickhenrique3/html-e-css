Principais Formataçoes

Negrito / Destaque

Nesta frase temos um termo em negrito basta usar a tag <B>(nao Sematica) e colocar entre as tags.

Nesta frase,tenho um Termo em destaque usando a tag <STRONG> (nao Sematica).

Itálico / Êfase

Nesta frase, temos um termo em itálico basta usar a tag <i>(nao Sematica).

Nesta frase temos em Ênfase usando a tag <em> (nao Sematica).

Testo marcado

Podemos criar um texto marcado usando a tag <mark>.

Texto deletado

podemos marcar um texto excluido para indicar que ele deve ser lido mas nao considerado.com a a tag <del>:

Texto inserido

podemos marcar um texto inserido para dar uma Ênfase.e indicar que ele foi adicionado depois. com a tag <ins>

Existe tambem a tag <u>(Nao semantica ) para sublinhar

Texto subscrito

Para inserir coisas do tipo H2O usando a tag <sub>

Texto sobrescrito
